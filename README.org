Emacs major mode for [[https://archibus.com/][ARCHIBUS]] log files.

* Installing

Add the following to your =~/.emacs= config:

#+begin_src emacs-lisp
(let ((ablog-mode-lib "~/path/to/emacs-archibus-log-mode/archibus-log.el"))
  (when (locate-library ablog-mode-lib)
    (load-library ablog-mode-lib)))
#+end_src

and restart your favorite editor.

* Features

- SQL syntax highlighting
  + ~SELECT~ in yellow
  + ~INSERT~ in green
  + ~UPDATE~ in blue
  + ~DELETE~ in red
- Commands for moving
  + Go to previous/next "read" SQL statement (~SELECT~)
  + Go to previous/next "change" SQL statement (~INSERT~, ~UPDATE~ or ~DELETE~)
  + Go to previous/next error
